# Wintoto

[Wintoto](http://www.win-toto.sg/ "Wintoto") is a website for learning patterns of Singapore Toto.

Web Client:

- angular-cli
- Echarts.js
- lodash

Web Server:

- Express(express-generator)
- MongoDB

Backend Server:
- python multi-thread crawler
- Selenium
- MongoDB
